# What is it

User script to run with a browser user-scripts runner (tampermonkey/UserScript/etc.)
It adds a button to the video player on coursera "leacture" pages which enables to pin the plyer in it's notmal size when scrolling down the page to see more content.

# How to install

- Add "tampermonkey" extension to your browser ("User Scripts" for Safari). And make sure to [allow user scripts execution](https://www.tampermonkey.net/faq.php#Q209).
- Open extension and go to `Utilities` tab.
- Ad `https://raw.githubusercontent.com/reoberon/pin-coursera-video/main/script.js` to `Import from URL` and click `Install`
- Voila
